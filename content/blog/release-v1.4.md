# 简介

正如标题所示，我们正式发布了1.4版本的DiscuzHub

## 原因

+ 更新的内容和代码较多
+ <del>作者手贱Version Code多写了一个版本</del>

## 主要更新内容

+ 对于发帖附件的支持
+ 对于发送密码贴的支持
+ 全屏模式支持广色域图片显示


## 修复了下面的BUG

+ 附件页因为缓存原因，没有及时消去信息

## 优化了

+ 图文混排时帖子卡顿的情况

## 下个版本优化对象

+ 添加论坛时界面优化
+ 首页界面的更新

## 已知BUG

+ 附件无法随着草稿的保存而保存
