论坛添加指引

感谢使用Discuz Hub, 本页面将会指引你添加论坛。

要求

Discuz Hub 要求您需要添加的论坛满足以下条件：

  • 论坛需要以Discuz X为架构
  • Discuz版本大于X 3.2
  • 移动端API版本大于4
  • 打开论坛移动页面并关闭防采集
  • 关闭其他可能会影响API访问的插件（已知的包括千帆网的插件）

您是网站拥有者？

您可以在Discuz！兼容性设置向导里找到兼容我们应用的方法。

步骤

1. 检查您的系统

请注意我们的应用目前仅支持Android/安卓平台，并且要求Android版本大于5.0，请检查您的系统是否符合要求。

我们的应用尽管兼容TV和平板，但是我们建议您在手机上使用Discuz Hub以获得更好的用户体验。

2.下载并安装Discuz Hub

您可以选择在应用市场上下载本应用。在相关的应用市场上搜索Discuz Hub下载并安装后即可使用。

中国大陆区应用变更说明

对于中国大陆用户而言，因为相关政策和法律法规，您在大陆的应用市场上下载的Discuz Hub会屏蔽某些功能或者模块。详情请参考此博客。

您可以使用首页的Github Release或者微云链接来下载本应用。下载应用后，安装APK文件即可。

权限提醒

请检查你所安装的Discuz Hub是否有除以下权限以外的要求。如果应用要求了敏感权限，请立即停止安装APK。

  • android.permission.INTERNET： 连接到网络 （我们需要此权限访问论坛API以完成正常的数据渲染和应用交互）
  • android.permission.ACCESS_NETWORK_STATE：获取网络状态 （我们需要此权限判定您是否处于计费流量下以选择图片等耗流量的资源是否下载）
  • android.permission.VIBRATE：振动手机 （我们需要此权限提醒您的操作是否合法）
  • （仅在低于Android 10/Q版本要求）android.permission.WRITE_EXTERNAL_STORAGE：读写外部数据 （我们需要此权限以完成图片或者附件的下载）

添加论坛 （版本低于2.1）

您可以通过以下步骤新增一个论坛。

  1. 打开Discuz Hub
  2. 点击右下角的加号
  3. 输入论坛的网址（通常就是论坛的域名，例如西北工业大学三航四方论坛->bbs.nwpu.edu.cn，请务必带上HTTP或者HTTPS一些）
    1. 输入网站的协议，如 http:// 或者 https://
    2. 接着输入网站的域名，如西北工业大学三航四方 bbs.nwpu.edu.cn
    3. 确定论坛下是否有一个前缀路径，例如一亩三分地会有BBS的路径
  4. 等待下方智能联想，当出现对号时，选择该路径
  5. 点击完成，经过app检测后即可新增论坛

智能添加方法

  1. 打开Discuz Hub
  2. 点击右下角的加号
  3. 向对话框直接复制一个来自论坛的帖子地址
  4. 等待下方智能联想，当出现对号时，选择该路径
  5. 点击完成，经过app检测后即可新增论坛
