# 简介

正如标题所示，我们正式发布了2.0版本的DiscuzHub，这是一个大版本更新，我们非常推荐用户升级到这个新的版本。

## 下载链接 v2.0

[Github 发布](https://github.com/kidozh/DiscuzHub/releases/download/v2.0/app-release.apk)

[微云下载](https://share.weiyun.com/xQvLw5GX)

## 主要更新内容

+ 采用抽屉式的论坛管理方式
+ 支持了回复时的发帖高级功能
+ 支持子论坛的查看
+ 修复了外链导致图片无法正常加载的问题
+ 新增历史浏览记录功能（默认关闭）
